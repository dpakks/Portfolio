# Deepak Kumar — Portfolio

My personal portfolio website showcasing my work, skills, and experience as a Cloud/DevOps Engineer and Software Developer.

## Sections

- **Home** — Introduction and tagline
- **About Me** — Background and what I do
- **Education** — Northeastern University (MS) & Sri Venkateswara College of Engineering (BE)
- **Skills** — Tech stack across Cloud, Backend, Frontend, and Databases
- **Experience** — Professional roles at Prodex Technologies and HealthOneCloud
- **Projects** — Featured work including cloud infrastructure, AI tools, and web apps

## Built With

- HTML, CSS, JavaScript
- [AOS](https://michalsnik.github.io/aos/) — Scroll animations
- [Iconify](https://iconify.design/) — Tech stack icons

## Run Locally

1. Clone the repo and open the folder in VS Code
2. Open `index.html` directly in a browser, or use the **Live Server** extension for hot reload
